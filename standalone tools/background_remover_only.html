Background Remover

Entfernt Hintergruende direkt im Browser und behaelt die Originalaufloesung. Exportiert wird als PNG ohne zusaetzliche Qualitaetsverluste.

Freistellung

Qualitaet bleibt erhalten, weil das Modell mit dem Originalbild arbeitet und das Ergebnis als PNG mit denselben Pixelmassen gespeichert wird. Die Vorschau ist nur eine verkleinerte Anzeige.

Export

Exportiert immer das freigestellte PNG-Ergebnis direkt aus dem erzeugten Blob. Kein erneutes JPEG-Rendering, kein Downscaling.

Freistellungs-Vorschau

Links Original, rechts das freigestellte Ergebnis. Pro Karte bleibt die originale Pixelgroesse sichtbar.
